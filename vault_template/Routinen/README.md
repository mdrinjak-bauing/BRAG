# Routinen — deine wiederkehrenden Aufgaben

Kurze Vorlagen für Dinge, die immer gleich ablaufen. Du nennst eine beim Namen
(„hol mich auf den Stand", „aktualisier das Quellenverzeichnis"), und Claude liest
die passende Datei und führt die Schritte mit den BRAG-Werkzeugen aus.

**Eigene anlegen:** neue `.md` hier ablegen — der Titel = das Thema, dazu ein
**Auslöser** (welche Worte die Routine starten) und nummerierte Schritte mit den
Tools. Ideen: „Tagebucheintrag schreiben", „diese Liste überarbeiten",
„Wochenrückblick erstellen", „Protokoll aus den Notizen ziehen".

Damit Claude die Routinen kennt, steht in der `CLAUDE.md`: *Routine-Aufgaben liegen
in `WissensWIKI/Routinen/` — nenne ich eine, lies die Datei und folge ihr.*

(Liegt im WissensWIKI — wird **nicht** durchsucht/indexiert.)
